# ADR-060 - ArchiMate Mapping for the Reference Architecture

## Status
Accepted

## Context
The repository explicitly maps logical services and implementation assets to ArchiMate layers and viewpoints.

## Decision
The architecture finalization work package incorporates this concern as a formal repository artifact rather than a side note.

## Consequences
- The release pack can be reviewed independently of implementation details.
- Future changes can be traced to a stable release baseline.
- The repository supports controlled evolution after the first stable release.
