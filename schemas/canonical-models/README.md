# Canonical Models Schemas

This directory contains the machine-readable schemas for the platform canonical model.

## Schemas

- execution-context.schema.json
- execution-request.schema.json
- execution-response.schema.json
- capability-definition.schema.json
- agent-descriptor.schema.json
- tool-descriptor.schema.json
- policy-decision.schema.json
- telemetry-event.schema.json
- audit-event.schema.json

## Usage

These schemas are used by orchestration services, agent gateways, tool adapters, observability pipelines, and governance workflows.
