# OCI Mapping

Reference OCI services for operations and platform engineering:

- OCI DevOps for pipeline orchestration
- OCI Resource Manager for infrastructure as code
- OCI Logging for operational and audit logs
- OCI Monitoring for metrics and alarms
- OCI Notifications for operational alerts
- OCI Object Storage for backups and retained artifacts
- OCI Bastion for controlled access
- OCI Vault for secrets and keys
- OCI Audit for administrative traceability

This chapter keeps the logical operations model separate from the product implementation choices.
