# Model, Prompt, and Tool Governance

Models, prompts, and tool definitions must be managed as governed artifacts.

## Model governance

- model selection criteria
- approved model catalog
- environment restrictions
- evaluation and revalidation

## Prompt governance

- prompt versioning
- approved system prompts
- prompt templates
- prompt reviews and change control

## Tool governance

- tool ownership
- allowed operations
- data scope
- connector trust level
- version control
