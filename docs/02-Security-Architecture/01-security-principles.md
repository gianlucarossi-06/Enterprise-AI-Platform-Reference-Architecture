# Security Principles

- Zero Trust verification at every hop.
- Least privilege for every token and service account.
- Delegated identity rather than credential forwarding.
- Policy as code rather than ad hoc checks.
- Audit everything that grants, denies, or escalates access.
- Separate authentication from authorization.
- Separate business capability permission from tool and resource permission.
- Down-scope access as the request moves deeper into execution.
