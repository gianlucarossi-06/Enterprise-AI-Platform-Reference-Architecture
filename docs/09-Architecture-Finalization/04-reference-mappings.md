# Reference Mappings

## C4 mapping

| C4 view | EARA view |
|---|---|
| System context | User Experience Plane and external enterprise resources |
| Container view | Control Plane, Execution Plane, and integration boundaries |
| Component view | Orchestrator, registry, security, observability, gateway, executors |
| Code view | Implementation details for individual services and adapters |

## TOGAF mapping

| TOGAF domain | EARA content |
|---|---|
| Business architecture | Business capabilities, governance, roles, and review outcomes |
| Application architecture | Orchestrator, registry, gateway, runtime, adapters |
| Data architecture | Canonical models, telemetry, audit, release metadata |
| Technology architecture | OCI services, OKE, networking, logging, vault, IAM |

## ArchiMate mapping

| ArchiMate layer | EARA representation |
|---|---|
| Business layer | Business capabilities and governance controls |
| Application layer | Platform services and agent execution services |
| Technology layer | OCI services, networking, containers, identity, storage |
| Motivation layer | Principles, requirements, and ADRs |
| Implementation layer | Volumes, diagrams, schemas, and release artifacts |
